## haxm-test

A bit of code to test [HAXM](https://github.com/intel/haxm) behaviour.

To clone, make and run from [MSYS2/MINGW](https://www.msys2.org/):


```
git clone https://github.com/JayFoxRox/haxm-test.git
cd haxm-test
mkdir build
cd build
cmake .. -G 'MSYS Makefiles'
make
./haxm-test
```
